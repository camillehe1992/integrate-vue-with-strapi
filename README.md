# integrate-vue-with-strapi

Build a project using Vue.js for the frontend and Strapi as the backend.

## Development

```sh
cd strapi
npm run develop  # The admin panel of Strapi runs at http://localhost:1337/admin

cd ../vue-app
npm run serve
```
